> [!date] Дата
> 12 июля 2026

## Цель

- Разделить музыкальный трек и трек паузы на отдельные `AudioSource`
- Реализовать корректную паузу/возобновление трека паузы
- Убрать дублирующий запуск трека паузы
- Добавить `fadeInDuration`, `fadeOutDuration`, `stopOnPause` в `SoundDefinition`

---

## Проблема

### 1. Трек паузы не останавливался при выходе из паузы

**Симптом:** При выходе из паузы (`ResumeMusic()`) трек паузы (`UiPauseOpen`) продолжал играть поверх музыкального трека.

**Причина:** Трек паузы запускался **дважды**:
- `PauseMusic()` → `PlayPauseSound()` → `_pauseSource` (основной трек)
- `AudioService.OnNavigationChanged()` → `Play("UiPauseOpen")` → пул `_uiVoices` (дубликат)

При выходе из паузы `ResumeMusic()` останавливал только `_pauseSource`, а дублирующий трек из пула продолжал играть.

### 2. Нет отдельного трека паузы

**Симптом:** При паузе музыка просто ставилась на паузу, но не было отдельного трека для паузы.

**Решение:** Создан `_pauseSource` — отдельный `AudioSource` для трека паузы. Работает по тому же принципу, что и `_musicSource`.

## Решение

### 1. Добавлены поля в `SoundDefinition`

**Файл:** `Assets/_Projects/Code/Futboloid.Core/Audio/SoundDefinition.cs`

```csharp
[Header("Fade")]
[SerializeField] private float fadeInDuration;
[SerializeField] private float fadeOutDuration;
[SerializeField] private bool stopOnPause;

public float FadeInDuration => fadeInDuration;
public float FadeOutDuration => fadeOutDuration;
public bool StopOnPause => stopOnPause;
```

### 2. Добавлено поле `pauseSoundId` в `AudioManager`

**Файл:** `Assets/_Projects/Code/Futboloid.Main/Audio/AudioManager.cs`

```csharp
[SerializeField] private string pauseSoundId = "UiPauseOpen";
```

### 3. Добавлен `_pauseSource`

**Файл:** `Assets/_Projects/Code/Futboloid.Main/Audio/AudioManager.cs`

```csharp
private AudioSource _pauseSource;
private CancellationTokenSource _pauseFadeCts;
private bool _pauseSoundPlaying;
private float _pauseFadeOutDuration = 1f;
```

В `Awake()`:
```csharp
_pauseSource = CreateSource("Pause", transform);
_pauseSource.loop = true;
_pauseSource.volume = 0f;
_pauseSource.playOnAwake = false;
```

### 4. Переделана логика `PauseMusic()` и `ResumeMusic()`

**PauseMusic():**
1. Останавливает SFX с `stopOnPause = true`
2. Ставит `_musicSource` на паузу через `Pause()`
3. Запускает `_pauseSource` с `PlayPauseSound()`

**ResumeMusic():**
1. Ставит `_pauseSource` на паузу через `Pause()`
2. Снимает `_musicSource` с паузы через `UnPause()`

### 5. Убран дублирующий вызов `Play("UiPauseOpen")`

**Файл:** `Assets/_Projects/Code/Futboloid.Core/Audio/AudioService.cs`

Убран блок:
```csharp
case NavigationState.Pause when e.Previous != NavigationState.Pause:
    _audio.Play(AudioCatalog.Ids.UiPauseOpen);
    break;
```

Теперь трек паузы запускается **только** через `_pauseSource` в `PauseMusic()`.

### 6. Добавлена обработка `ResumeMusic()` при выходе из паузы

**Файл:** `Assets/_Projects/Code/Futboloid.Core/Audio/AudioService.cs`

```csharp
// Восстановить музыку при выходе из паузы
if (e.Current != NavigationState.Pause && e.Previous == NavigationState.Pause && _onField)
    _audio.ResumeMusic();
```

### 7. Добавлено `stopOnPause` в `SfxVoice`

**Файл:** `Assets/_Projects/Code/Futboloid.Main/Audio/AudioManager.cs`

```csharp
private sealed class SfxVoice
{
    public bool StopOnPause { get; set; }
    // ...
}
```

При паузе вызывается `StopSfxWithStopOnPause()`, который останавливает все звуки с `StopOnPause = true`.

### 8. Обновлён `AudioCatalog.asset`

Для всех звуков добавлены поля:
```yaml
fadeInDuration: 0
fadeOutDuration: 0
stopOnPause: 0
```

Для `MusicMatch` и `UiPauseOpen` установлены значения:
```yaml
fadeInDuration: 0
fadeOutDuration: 1.5
```

## Итоговая архитектура

```
AudioManager
├── _musicSource (Music)
│   ├── loop: true
│   ├── fadeDuration: 1.5
│   └── fadeOutDuration: 1.5
├── _pauseSource (Pause)
│   ├── loop: true
│   └── fadeDuration: 2 (UiPauseOpen)
├── _sfxVoices (SFX)
│   └── StopOnPause (bool)
└── _uiVoices (UI)
    └── StopOnPause (bool)
```

---

## Логика работы

```
Нажатие Escape (пауза):
  ↓
PauseMusic()
  ├── StopSfxWithStopOnPause() — останавливает SFX
  ├── _musicSource.Pause() — музыка на паузе
  └── PlayPauseSound() — запускает трек паузы

Снятие с паузы:
  ↓
ResumeMusic()
  ├── _pauseSource.Pause() — трек паузы на паузе
  └── _musicSource.UnPause() — музыка возобновляется

Полный выход из игры (запуск нового трека):
  ↓
PlayMusic()
  ├── _pauseSource.Stop() — трек паузы останавливается
  └── _musicSource.Play() — музыкальный трек
```
