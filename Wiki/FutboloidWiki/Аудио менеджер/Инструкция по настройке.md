# Инструкция по настройке звуков

> [!date] Обновлено
> **05.07.2026** — Новая архитектура: AudioCatalog + AudioPlaybackHost на Root.

---

## Быстрый старт

### 1. Root.unity — хост воспроизведения

1. Открой сцену `Root.unity`
2. Создай пустой объект `Audio` (можно дочерний к корню)
3. Add Component → `AudioPlaybackHost`
4. В Inspector:
   - **Sfx Pool Size:** `8`
   - **Ui Pool Size:** `3` (можно оставить, UI звуков пока нет)
   - **Enable Fade:** включено

> [!important] Без этого объекта DI выдаст ошибку при старте — `RegisterComponentInHierarchy` ищет `AudioPlaybackHost` на Root.

### 2. AudioCatalog — CMS звуков

Asset: `Assets/_Projects/Resources/Data/Settings/AudioCatalog.asset`

Создание: **Assets → Create → Futboloid → Audio Catalog**

Каждая строка в **Sounds** — один тип звука:

| Поле | Пример BallHit | Пример MusicMatch |
|------|------------------|-------------------|
| **Id** | `BallHit` | `MusicMatch` |
| **Channel** | `GameplaySfx` | `Music` |
| **Clips** | 7 файлов из `Audio/SFX/Ball/` | 3 файла из `Audio/Music/` |
| **Mixer Group** | `ball` | `Traks` |
| **Priority** | `80` | `0` |
| **Allow Overlap** | ✅ | ❌ |
| **Cooldown** | `0.2` | `0` |
| **Loop** | ❌ | ✅ |
| **Enable Fade** | ❌ | ✅ |
| **Fade Duration** | `0` | `1.5` |

> [!howto] Mixer Group
> 1. Открой `Assets/_Projects/Audio/Mixers/NewAudioMixer 1.mixer`
> 2. Перетащи нужную группу (`ball`, `start`, `Crown`, `Traks`) в поле **Mixer Group**

### 3. Game.unity — убрать старое

Удали объект `AudioManager` и все дочерние `Speaker_*` — они больше не используются.

## Каталог Sound ID

| Id | Описание | Mixer (рекомендация) |
|----|----------|----------------------|
| `BallHit` | Удар мяча | `ball` |
| `GoalScored` | Гол забит | `Crown` |
| `GoalConceded` | Гол пропущен | `Crown` |
| `MatchStart` | Свисток начала | `start` |
| `MatchEnd` | Свисток конца | `start` |
| `MusicMatch` | Музыка матча | `Traks` |
| `PerkPick` | Выбор бонуса | `ball` или UI (позже) |

---

## Проверка в Play Mode

- [ ] Удар о стену / защитника — `BallHit`
- [ ] Гол / пропуск — `GoalScored` / `GoalConceded`
- [ ] Подача — свисток `MatchStart`
- [ ] Старт таймера — музыка `MusicMatch`
- [ ] Конец матча — `MatchEnd`, музыка затухает
- [ ] Reset поля — музыка останавливается
- [ ] Выход в меню — музыка на паузе
- [ ] Возврат на поле — музыка продолжается
- [ ] Выбор perk — `PerkPick`

---

## Добавить новый звук

1. Добавь строку в `AudioCatalog` с уникальным **Id**
2. Если нужен новый триггер — добавь событие на шину и подписку в `AudioService`
3. **Не** вызывай звук из геймплей-кода напрямую
