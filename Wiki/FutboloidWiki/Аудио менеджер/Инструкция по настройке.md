# Инструкция по настройке звуков

> [!date] Обновлено
> **08.07.2026** — `AudioManager` + конфиг в Inspector, музыка на `OnField`.

---

## Быстрый старт

### 1. Root.unity — AudioManager

1. Открой сцену `Root.unity`
2. Создай пустой объект `Audio` (можно дочерний к корню)
3. Add Component → `AudioManager`
4. В Inspector:
   - **Config** → `Assets/_Projects/Resources/Data/Settings/AudioCatalog.asset`
   - **Sfx Pool Size:** `8`
   - **Ui Pool Size:** `3`
   - **Enable Fade:** включено

> [!important] Без этого объекта DI выдаст ошибку — `RegisterComponentInHierarchy` ищет `AudioManager` на Root.

### 2. AudioCatalog — CMS звуков

Asset: `Assets/_Projects/Resources/Data/Settings/AudioCatalog.asset`

Создание: **Assets → Create → Futboloid → Audio Catalog**

Каждая строка в **Sounds** — один тип звука:

| Поле | Пример BallHit | Пример MusicMatch |
|------|----------------|-------------------|
| **Id** | `BallHit` | `MusicMatch` |
| **Channel** | `GameplaySfx` | `Music` |
| **Clips** | 7 файлов из `SFX/Ball/` | 3 файла из `Audio/Music/` |
| **Mixer Group** | `ball wall` | `Traks` |
| **Priority** | `80` | `0` |
| **Allow Overlap** | ✅ | ❌ |
| **Cooldown** | `0.2` | `0` |
| **Loop** | ❌ | ✅ |
| **Enable Fade** | ❌ | ✅ |
| **Fade Duration** | `0` | `1.5` |
| **Base Pitch** | `1` | `1` |
| **Pitch Random Range** | `0.08` | `0` |

> [!howto] Mixer Group
> 1. Открой `Assets/_Projects/Audio/Mixers/NewAudioMixer 1.mixer`
> 2. Перетащи группу (`ball`, `ball wall`, `start`, `Crown_Scored`, `Traks`, …) в **Mixer Group**

### 3. Game.unity — убрать старое

Удали объект `AudioManager` (legacy) и все дочерние `Speaker_*`, если ещё остались.

## Проверка в Play Mode

- [ ] Вход на поле — музыка `MusicMatch` (до подачи мяча)
- [ ] Подача — только свисток `MatchStart`
- [ ] Удар о стену — `BallHit`
- [ ] Удар о защитника/вратаря — `BallHitMan`
- [ ] Гол / пропуск — `GoalScored` / `GoalConceded`
- [ ] Escape (пауза) — музыка затухает
- [ ] Continue — музыка с того же места
- [ ] Выход в главное меню — музыка затухает
- [ ] Возврат на поле — музыка с того же места
- [ ] Рестарт турнира на поле — новый трек
- [ ] Конец матча — `MatchEnd`, музыка стоп
- [ ] Очки / комбо / бафф — звуки (если клипы назначены)

---

## Добавить новый звук

1. Положи файлы в `Assets/_Projects/Audio/`
2. Добавь строку в `AudioCatalog` с уникальным **Id**
3. Если нужен новый триггер — событие на шине + подписка в `AudioService`
4. Для UI-кнопок — `IAudioManager.Play(id)` через DI
5. **Не** вызывай `AudioSource` из геймплей-кода напрямую
