# Инструкция по настройке звуков

> [!date] Обновлено
> **05.07.2026** — Свисток при начале матча, пауза музыки при выходе в меню.

---

## Быстрый старт

### 1. Открой сцену `Game.unity`

В Hierarchy найди объект `AudioManager`.

### 2. Создай динамики

Каждый тип звука — это отдельный дочерний объект внутри `AudioManager`.

**Шаги:**
1. Правой кнопкой на `AudioManager` → `Create Empty`
2. Назови по имени события: `Speaker_BallHit`, `Speaker_Goal`, `Speaker_Start`
3. На каждый объект повесь скрипт `AudioClipSource`

### 3. Настрой каждый динамик

В Inspector каждого `AudioClipSource` заполни:

| Поле | Что делать |
|------|------------|
| **Event Name** | Имя события (например, `BallHit`) |
| **Clips** | Перетащи массив звуков (например, 5 ударов) |
| **Mixer Group** | **Важно!** Выбери группу микшера (например, `SFX` или `Music`), чтобы видеть индикацию |
| **Priority** | 0-10 (важные), 50 (средние), 80-100 (частые) |
| **Allow Overlap** | False для голосовых, True для SFX |
| **Cooldown** | Минимальный интервал между звуками (0 = без ограничения) |
| **Loop** | True для музыки, False для всех остальных звуков |
| **Enable Fade** | True для плавного появления/затухания |
| **Fade Duration** | Длительность фейда в секундах (обычно 0.5-1.5) |

### 4. Глобальный тумблер фейда

В `AudioManager` на сцене есть поле `Enable Fade`. Выключите его, если хотите мгновенное воспроизведение/остановку всех звуков.

## Каталог событий для настройки

| Динамик | Event Name | Описание | Где вызывается (Код) |
|---------|------------|----------|---------------------|
| `Speaker_BallHit` | `BallHit` | Удар о стену/защитника/вратаря | `BallMotion.ResolveHit()` |
| `Speaker_Goal` | `GoalScored` | Гол забит | `BallMotion.TryScoreGoal()` |
| `Speaker_GoalConceded` | `GoalConceded` | Гол пропущен | `BallMotion.TryScoreGoal()` |
| `Speaker_Start` | `MatchStart` | Свисток перед матчем | `BallMotion.Serve()` |
| `Speaker_End` | `MatchEnd` | Свисток конца матча | `MatchFlow.EndMatch()` |
| `Speaker_Perk` | `PerkPick` | Выбор бонуса | `RunStateService.ApplyPerkPick()` |
| `Speaker_Music` | `MusicStart` | Музыка матча (loop + fade) | `MatchFlow.OnBallServed()` |

## Настройка музыки

Для музыки создай отдельный динамик `Speaker_Music`:
1. Создай дочерний объект в `AudioManager`
2. Назови: `Speaker_Music`
3. В `AudioClipSource` установи:
   - **Event Name:** `MusicStart`
   - **Clips:** 1-2 трека (будет выбран случайный)
   - **Loop:** `True` (обязательно!)
   - **Mixer Group:** `Music`
   - **Priority:** `0` (чтобы не прерывалась)
   - **Cooldown:** `0`
   - **Allow Overlap:** `False`
   - **Enable Fade:** `True`
   - **Fade Duration:** `1.0` или `1.5`

**Логика:** Музыка запускается автоматически при подаче мяча. При рестарте матча (`Reset`) или окончании (`EndMatch`) музыка плавно затухает и останавливается.

---

## Пауза музыки

При выходе в главное меню музыка ставится на паузу и возобновляется при возврате в игру.

**Как работает:**
1. Выход в меню → `AudioManager.PauseMusic()` — музыка ставится на паузу (позиция сохраняется)
2. Возврат в игру → `AudioManager.ResumeMusic()` — музыка продолжается с того же места

**Не нужно настраивать** — это работает автоматически.
