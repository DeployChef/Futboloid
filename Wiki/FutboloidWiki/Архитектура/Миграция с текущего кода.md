# Миграция с текущего кода

← [[Индекс архитектуры]] | [[Обзор архитектуры]]

> [!date] Статус
> **05.07.2026** — миграция **завершена** для MVP-скелета. Ниже — что было и что стало.

## Было → стало

| Было | Стало |
|------|-------|
| `MainMenu.unity` + `MainMenuController` | `MainMenuWidget` на `Root.unity` (оверлей) |
| `GameManager` singleton | `MatchFlow` + `PitchStateMachine` |
| `SampleScene` | `Game.unity` (additive) |
| `AudioManager` на Game-сцене (legacy) | `AudioService` + `AudioManager` на Root |
| Build: MainMenu → SampleScene | Build: только `Root.unity` |
| `GameOld.unity` | удалена |

## План миграции (этапы)

### Этап 1 — Startup + Root

- [x] `Root.unity`, VContainer + UniTask
- [x] `Startup`, `GameDirector`, `AppRootState`
- [x] Build settings: только Root
- [x] `RootLifetimeScope` + `RegisterRootScope`

### Этап 2 — Game additive

- [x] `Game.unity`
- [x] `AppGameState` грузит сцену additive
- [x] `RegisterComponentInScene` + `InjectGameObject` — см. [[Связь сцены с кодом]]
- [x] Match HUD / Run XP / Tournament / BonusPick на `Game.unity`

### Этап 3 — FSM

- [x] `OverlayStateController`: MainMenu / OnField / Pause / Tournament
- [x] `PitchStateMachine`: KickoffWait / Simulating / Reshuffle / BonusPick / MatchEnded
- [x] `GameManager` разобран на `MatchFlow` + `PitchStateMachine`

### Этап 4 — Main Menu overlay

- [x] `MainMenu.unity` убрана из билда
- [x] `MainMenuWidget` + `PauseMenuView` на Root
- [ ] `BotSimulationController` для живого фона меню — **post-MVP** (сейчас заморозка, см. [[UI и оверлеи]])

### Этап 5 — Polish

- [ ] Scene transition (турнир)
- [ ] `ISaveStorage` + лидерборд
- [ ] Tournament polish

## Маппинг GameManager → новые классы

| Было в GameManager | Куда |
|--------------------|------|
| `GameState` enum | `PitchStateMachine` |
| Таймер матча | `MatchFlow` |
| Счёт голов | `MatchFlow` |
| `RecordGoal` | `GoalScoredEvent` → Pitch FSM → Reshuffle |
| `StartMatch` LoadScene | убрано — сцена уже загружена |
| Singleton Instance | убрано — DI |
