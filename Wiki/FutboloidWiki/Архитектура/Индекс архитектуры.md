## Диаграмма состояний (краткая)

```mermaid
stateDiagram-v2
    state "App" as app {
        Startup --> AppRoot
        AppRoot --> AppGame
    }

    state "Navigation" as nav {
        MainMenu --> OnField: Играть
        OnField --> Pause: Escape
        Pause --> OnField: Продолжить
        OnField --> Tournament: Матч кончен
        Tournament --> OnField: След. матч
        Pause --> MainMenu: В меню
    }

    state "Pitch" as pitch {
        KickoffWait --> Simulating: Пробел
        Simulating --> Reshuffle: Гол
        Reshuffle --> KickoffWait: Done
        Simulating --> MatchEnded: 90с или вайп врагов
    }

    AppGame --> OnField: cold start (isColdStart)
    MainMenu --> OnField: Играть
    OnField --> KickoffWait
```
