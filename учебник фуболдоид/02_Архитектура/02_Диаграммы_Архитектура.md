## 🚀 Диаграмма запуска приложения

```mermaid
sequenceDiagram
    participant User as Пользователь
    participant Startup as Startup.cs
    participant DI as VContainer
    participant Director as GameDirector
    participant App as AppRootState
    participant Overlay as OverlayStateController
    
    User->>Startup: Запуск игры
    Startup->>Startup: Awake()
    Startup->>DI: rootScope.Build()
    DI-->>Startup: Контейнер готов
    Startup->>DI: Resolve<IGameDirector>()
    DI-->>Startup: GameDirector
    Startup->>Director: InitializeGameAsync()
    Director->>App: new AppRootState()
    App->>App: Enter()
    Director->>App: Enter()
    App->>Overlay: Create overlay
    Director->>Director: Infrastructure ready
    Startup->>Startup: FadeOutLoadingAsync()
    Startup-->>User: Игра запущена
```
