## 📈 Метрики DI

| Метрика | Значение | Описание |
|---------|----------|----------|
| Singleton-сервисов | 10+ | EventBus, AudioManager, и др. |
| Transient-сервисов | 5+ | DefenderLogic, Temporary objects |
| Компонентов из сцены | 8+ | BallView, GoalkeeperView, и др. |
| LifetimeScope | 2 | Root, Game |
| Строки кода | ~300 | Все LifetimeScope файлы |

graph TD

    Root --> |Регистрирует| EB[EventBus]
    Root --> |Регистрирует| AR[AppRootState]
    
    Root --> |При переходе на Game| GS[GameScope]
    GS --> |Регистрирует| BM[BallMotion]

    GS --> |Регистрирует| DL[DefenderLogic]
    GS --> |Из сцены| BV[BallView]
    GS --> |Из сцены| GV[GoalkeeperView]
    GS --> |Из сцены| DS[DefenderSpawner]
    
    Root -.->|Singleton| AM
    Root -.->|Singleton| EB
    GS -.->|Singleton| BM
    GS -.->|Singleton| MF
    GS -.->|Transient| DL

## 🔄 Диаграмма жизненного цикла объектов

```mermaid
stateDiagram-v2
    [*] --> Registered: Регистрация в DI
    
    Registered --> Singleton: lifetime = Singleton
    Registered --> Transient: lifetime = Transient
    Registered --> SceneComponent: RegisterComponentInScene
    
    Singleton --> Active: Resolve()
    Transient --> Active: Resolve()
    SceneComponent --> Active: Найти в сцене
    
    Active --> Destroyed: DestroyScene() или Dispose()
    Destroyed --> [*]
    
    note right of Singleton
        Живёт до конца игры
        AudioManager, EventBus
    end note
    
    note right of Transient
        Создаётся каждый раз
        DefenderLogic
    end note
    
    note right of SceneComponent
        Живёт в сцене
        BallView, GoalkeeperView
    end note
```
