## 🚀 КАК УЧИТЬСЯ

### Правильный подход

```
1. Читаешь шаг
2. Понимаешь теорию
3. Пишешь код по примеру
4. Делаешь практику
5. Повторяешь через день
```

### Не правильный подход

```
❌ Пропускать шаги
❌ Не писать код
❌ Торопиться
❌ Забывать повторять
```
