## 5.1 RunProgressionService

### Что такое RunProgression?

**Run** — это один забег (от начала до конца турнира). Внутри забега:
- Игрок проходит несколько матчей
- После каждого матча выбирает перк
- Перки дают постоянные бонусы на весь забег
- При смерти забег заканчивается, перки сбрасываются

### Ключевые сервисы

| Сервис | Файл | Роль |
|--------|------|------|
| `IRunProgressionService` | `Futboloid.Core/Run/IRunProgressionService.cs` | Интерфейс прогрессии |
| `RunProgressionService` | (в Main) | Реализация |
| `TournamentRunService` | `Futboloid.Main/Session/TournamentRunService.cs` | Управление забегами |
| `PerkDefinition` | `Futboloid.Core/Run/PerkDefinition.cs` | Конфиг перка |
| `PerkIds` | `Futboloid.Core/Run/PerkIds.cs` | ID перков |

### Что хранит RunProgressionService

```csharp
// Бонусы, действующие на весь забег
- GoalkeeperKickMultiplier  — сила удара вратаря
- GhostBallCharges          — заряды «сквозного» мяча
- И другие...
```

### Как получить значение

```csharp
// В BallMotion — проверяем силу удара вратаря
var kickMul = _runProgression?.GetGoalkeeperKickMultiplier() ?? 1f;
Speed = _settings.ServeSpeed * kickMul;
```

## 5.2 PerkDefinition и StatusEffects

### PerkDefinition

```csharp
public class PerkDefinition : ScriptableObject
{
    [SerializeField] private string id;
    [SerializeField] private string displayName;
    [SerializeField] private string description;
    [SerializeField] private PerkType type;
    [SerializeField] private int value;
}
```

### StatusEffects — временные эффекты

| Эффект | Когда действует | Длительность |
|--------|----------------|-------------|
| Buff | Бонусный эффект | На матч / на розыгрыш |
| Debuff | Штрафный эффект | На матч / на розыгрыш |

### StatusEffectService

```csharp
public interface IStatusEffectService
{
    void Apply(StatusEffectDefinition effect);
    void Remove(StatusEffectId effectId);
    bool HasActive(StatusEffectId effectId);
}
```
