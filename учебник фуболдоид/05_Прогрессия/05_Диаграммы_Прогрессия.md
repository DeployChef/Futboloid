## 🔄 Диаграмма прогрессии забега

```mermaid
graph TB
    subgraph "Tournament"
        Run[Run / Забег]
        Match1[Match 1]
        Perk1[Perk Pick 1]
        Match2[Match 2]
        Perk2[Perk Pick 2]
        Match3[Match 3]
    end
    
    subgraph "RunProgressionService"
        Level[RunLevel]
        XP[RunXP]
        Perks[Collected Perks]
    end
    
    subgraph "StatusEffects"
        Buff[Buff]
        Debuff[Debuff]
    end
    
    Run --> Match1
    Match1 --> Perk1
    Perk1 --> Match2
    Match2 --> Perk2
    Perk2 --> Match3
    
    Run --> Level
    Run --> XP
    Run --> Perks
    
    Perk1 --> Buff
    Perk2 --> Buff
    
    style Run fill:#FFE4B5
    style Match1 fill:#90EE90
    style Perk1 fill:#87CEEB
    style Level fill:#FFB6C1
```
