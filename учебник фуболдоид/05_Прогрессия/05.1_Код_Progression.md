## 🎯 Зачем нужна прогрессия?

**Прогрессия** — это система, которая:
1. Отслеживает уровень забега
2. Хранит перки, выбранные игроком
3. Дает бонусы за прогресс
4. Сбрасывается при смерти

## 🔍 Пошаговый разбор кода

### 1. Интерфейс IRunProgressionService

```csharp
public interface IRunProgressionService
{
    int RunLevel { get; }
    int RunXP { get; }
    
    float GetGoalkeeperKickMultiplier();
    int GetGhostBallCharges();
    
    void AddXP(int xp);
    void PickPerk(PerkDefinition perk);
    void Reset();
}
```

**Почему интерфейс?**
- Легко тестировать
- Можно подменить на тестовый
- DI-контейнер знает, что зарегистрировать

---

### 2. Структура RunProgressionService

```csharp
public class RunProgressionService : IRunProgressionService
{
    public int RunLevel { get; private set; }
    public int RunXP { get; private set; }
    
    private readonly Dictionary<string, int> _perks = new();
    private readonly List<PerkDefinition> _pickedPerks = new();
    
    public float GetGoalkeeperKickMultiplier()
    {
        if (!_perks.TryGetValue("GoalkeeperKick", out var level))
            return 1f;
        
        return 1f + level * 0.1f;  // +10% за уровень
    }
}
```

**Почему `Dictionary<string, int>`?**
- `string` — ID перка (например, "GoalkeeperKick")
- `int` — уровень перка
- **Альтернатива:** `Dictionary<string, PerkDefinition>` — но int проще

---

### 3. Метод AddXP()

```csharp
public void AddXP(int xp)
{
    RunXP += xp;
    
    // Проверяем, не достигнут ли новый уровень
    while (RunXP >= XPToNextLevel)
    {
        RunXP -= XPToNextLevel;
        RunLevel++;
        PublishLevelUp();
    }
}
```

**Почему `while`?**
- Можно получить много XP за раз
- Может быть несколько уровней за раз
- **Альтернатива:** `if` — но не учтёт несколько уровней

---

### 4. Метод PickPerk()

```csharp
public void PickPerk(PerkDefinition perk)
{
    var id = perk.Id;
    
    if (!_perks.TryGetValue(id, out var level))
        _perks[id] = 0;
    
    _perks[id]++;
    _pickedPerks.Add(perk);
    
    PublishPerkPicked(perk);
}
```

**Почему `_perks[id]++`?**
- Перк может быть выбран несколько раз
- Каждый уровень увеличивает бонус
- **Зачем?** Чтобы игрок мог качать перки

---

### 5. Метод Reset()

```csharp
public void Reset()
{
    RunLevel = 0;
    RunXP = 0;
    _perks.Clear();
    _pickedPerks.Clear();
    
    PublishReset();
}
```

**Почему `_perks.Clear()`?**
- При смерти все перки сбрасываются
- **Зачем?** Чтобы забег был уникальным

## ✅ Шаблон для копирования

```csharp
public class MyProgressionService : IMyProgressionService
{
    public int Level { get; private set; }
    public int XP { get; private set; }
    
    private readonly Dictionary<string, int> _upgrades = new();
    
    public void AddXP(int xp)
    {
        XP += xp;
        while (XP >= XPToNextLevel)
        {
            XP -= XPToNextLevel;
            Level++;
            OnLevelUp?.Invoke(Level);
        }
    }
    
    public void Upgrade(string upgradeId)
    {
        if (!_upgrades.TryGetValue(upgradeId, out var level))
            _upgrades[upgradeId] = 0;
        
        _upgrades[upgradeId]++;
        OnUpgrade?.Invoke(upgradeId, _upgrades[upgradeId]);
    }
    
    public void Reset()
    {
        Level = 0;
        XP = 0;
        _upgrades.Clear();
        OnReset?.Invoke();
    }
}
```

## 🚫 Антипаттерны

### 1. Забывать сбрасывать прогресс

```csharp
// ❌ ПЛОХО — перки остаются
public void Reset()
{
    RunLevel = 0;
    RunXP = 0;
    // Забыли _perks.Clear()!
}

// ✅ ХОРОШО — сбрасываем всё
public void Reset()
{
    RunLevel = 0;
    RunXP = 0;
    _perks.Clear();
    _pickedPerks.Clear();
}
```

### 2. Использовать if вместо while

```csharp
// ❌ ПЛОХО — не учтёт несколько уровней
if (RunXP >= XPToNextLevel)
{
    RunXP -= XPToNextLevel;
    RunLevel++;
}

// ✅ ХОРОШО — учтёт несколько уровней
while (RunXP >= XPToNextLevel)
{
    RunXP -= XPToNextLevel;
    RunLevel++;
}
```

### 3. Забывать публиковать события

```csharp
// ❌ ПЛОХО — никто не узнает об изменении
public void AddXP(int xp)
{
    RunXP += xp;
    RunLevel++;
    // Забыли PublishLevelUp()!
}

// ✅ ХОРОШО — публикуем события
public void AddXP(int xp)
{
    RunXP += xp;
    RunLevel++;
    PublishLevelUp();
}
```
