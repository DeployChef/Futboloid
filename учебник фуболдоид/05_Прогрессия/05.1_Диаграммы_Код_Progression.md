## 🔄 Диаграмма жизненного цикла перка

```mermaid
sequenceDiagram
    participant Player as Игрок
    participant BPC as BonusPickCoordinator
    participant RPS as RunProgressionService
    participant Bus as IGameEventBus
    participant Audio as AudioService
    
    Player->>BPC: Выбирает перк
    BPC->>RPS: PickPerk(perk)
    RPS->>RPS: _perks[perk.Id]++
    RPS->>RPS: _pickedPerks.Add(perk)
    RPS->>Bus: Publish(PerkPickedEvent)
    Bus->>Audio: Play(PerkPickSound)
    BPC->>Player: Перк применён
```

## 🔄 Диаграмма начисления XP

```mermaid
stateDiagram-v2
    [*] --> Normal
    Normal --> AddXP: AddXP(xp)
    AddXP --> CheckLevel: RunXP += xp
    CheckLevel --> LevelUp: RunXP >= XPToNextLevel
    LevelUp --> AddMoreXP: RunXP -= XPToNextLevel
    AddMoreXP --> CheckLevel
    CheckLevel --> Normal: RunXP < XPToNextLevel
    LevelUp --> Publish: PublishLevelUp()
    Publish --> Normal
    
    note right of Normal
        Обычное состояние
    end note
    
    note right of AddXP
        Начисление XP
    end note
    
    note right of LevelUp
        Новый уровень
        Звук level up
    end note
```
