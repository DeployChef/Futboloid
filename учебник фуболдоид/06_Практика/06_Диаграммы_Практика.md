## 🔄 Диаграмма добавления нового звука

```mermaid
sequenceDiagram
    participant Dev as Разработчик
    participant Audio as Audio/
    participant Cat as AudioCatalog
    participant AS as AudioService
    participant Bus as IGameEventBus
    participant GP as Gameplay Code
    
    Dev->>Audio: Создаю папку с файлами
    Dev->>Cat: Добавляю новую строку в Sounds
    Dev->>AS: Добавляю подписку
    Dev->>Bus: Создаю новое событие
    GP->>Bus: Publish(НовоеСобытие)
    Bus->>AS: OnNewEvent(event)
    AS->>Cat: Play(НовыйЗвук)
    Cat-->>Dev: Готово!
```

## 🔄 Диаграмма добавления новой UI-страницы

```mermaid
sequenceDiagram
    participant Dev as Разработчик
    participant View as SettingsView
    participant DI as DI Container
    participant Nav as OverlayStateController
    participant Bus as IGameEventBus
    
    Dev->>View: Создаю View
    Dev->>DI: Регистрирую в DI
    Dev->>Nav: Подключаю к навигации
    Dev->>Bus: Добавляю состояние
    Nav->>View: Show()
    View-->>Dev: Экран показан
```
