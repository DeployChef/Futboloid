## 🔄 Диаграмма столкновений мяча

```mermaid
sequenceDiagram
    participant Ball as BallMotion
    participant Physics as Physics2D
    participant Hit as RaycastHit2D
    participant Audio as AudioService
    participant Match as MatchFlow
    
    Ball->>Physics: CircleCast()
    Physics-->>Ball: RaycastHit2D
    
    alt Удар о стену
        Ball->>Hit: layer == Wall
        Ball->>Ball: ReflectFromHit()
        Ball->>Bus: Publish(BallContactEvent.Wall)
        Bus->>Audio: Play("BallHit")
    else Удар о вратаря
        Ball->>Hit: layer == Keeper
        Ball->>Ball: ReflectFromKeeperHit()
        Ball->>Ball: ApplyKeeperBoost()
        Ball->>Bus: Publish(BallContactEvent.Keeper)
    else Удар о защитника
        Ball->>Hit: layer == Defender
        Ball->>Defender: HandleBallContact()
        Ball->>Bus: Publish(BallContactEvent.Defender)
    else Гол
        Ball->>Physics: OverlapCircle(Goal)
        Ball->>Ball: Stop()
        Ball->>Bus: Publish(GoalScoredEvent)
        Bus->>Match: RecordGoal()
        Bus->>Audio: Play("GoalScored")
    end
```
