## 🔄 Диаграмма жизненного цикла мяча

```mermaid
stateDiagram-v2
    [*] --> ResetAt: ResetAt(position)
    ResetAt --> AttachToAnchor: AttachToAnchor()
    AttachToAnchor --> Serve: Serve(direction)
    Serve --> Tick: Tick(deltaTime)
    Tick --> Tick: Продолжаем движение
    Tick --> ResolveHit: Столкновение
    Tick --> TryScoreGoal: Гол
    ResolveHit --> Tick: Отскок
    TryScoreGoal --> [*]: Гол засчитан
    
    note right of ResetAt
        Мяч стоит на месте
        Позиция задана
    end note
    
    note right of Serve
        Вратарь подаёт мяч
        Направление задано
    end note
    
    note right of Tick
        Движение каждый кадр
        Проверка столкновений
    end note
    
    note right of ResolveHit
        Обработка столкновения
        Публикация события
    end note
```
