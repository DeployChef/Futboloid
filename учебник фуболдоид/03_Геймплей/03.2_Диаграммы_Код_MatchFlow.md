## 🔄 Диаграмма жизненного цикла матча

```mermaid
sequenceDiagram
    participant Game as MatchFlow
    participant Bus as IGameEventBus
    participant Audio as AudioService
    participant Timer as Timer Loop
    participant HUD as HUD Widget
    
    Game->>Bus: Subscribe<GoalScoredEvent>
    Game->>Bus: Subscribe<MatchEndedEvent>
    Game->>Bus: Subscribe<PitchPhaseChangedEvent>
    Game->>Bus: Subscribe<MatchTimeAdjustedEvent>
    Game->>Bus: Subscribe<BallServedEvent>
    
    loop Каждый кадр
        Timer->>Game: Tick(deltaTime)
        Game->>Game: RemainingSeconds -= deltaTime
        Game->>Bus: Publish(MatchTimerChangedEvent)
        Bus->>HUD: UpdateTimerDisplay()
    end
    
    alt Гол
        Game->>Bus: Publish(GoalScoredEvent)
        Bus->>Game: RecordGoal()
        Bus->>Audio: Play("GoalScored")
    end
    
    alt Конец матча
        Game->>Bus: Publish(MatchEndedEvent)
        Bus->>Audio: StopMusic()
        Bus->>Audio: Play("MatchEnd")
    end
```

## 🔄 Диаграмма состояний MatchFlow

```mermaid
stateDiagram-v2
    [*] --> Reset
    Reset --> TimerRunning: BallServedEvent
    TimerRunning --> TimerPaused: PitchPhase.BonusPick
    TimerPaused --> TimerRunning: PitchPhase.Simulating
    TimerRunning --> MatchEnded: RemainingSeconds <= 0
    MatchEnded --> [*]
    
    note right of Reset
        Сброс счета и таймера
        PlayerScore = 0
        OpponentScore = 0
    end note
    
    note right of TimerRunning
        Таймер тикает
        Очки считаются
    end note
    
    note right of TimerPaused
        Таймер на паузе
        Выбор перка
    end note
    
    note right of MatchEnded
        Матч окончен
        Результат записан
    end note
```
