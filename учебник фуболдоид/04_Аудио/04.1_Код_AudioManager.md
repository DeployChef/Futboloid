## 🎯 Зачем нужен AudioManager?

**AudioManager** — это сердце аудио-системы. Он:
1. Управляет пулами AudioSource (8 SFX, 3 UI, 1 music, 1 pause)
2. Воспроизводит звуки по SoundId
3. Управляет приоритетами (clip низкоприоритетных звуков)
4. Делает Fade In / Fade Out
5. Паузит и возобновляет музыку

## 🔍 Пошаговый разбор кода

### 1. Поля класса

```csharp
[SerializeField] private AudioCatalog config;
[SerializeField] private AudioMixer mixer;
[SerializeField] private int sfxPoolSize = 8;
[SerializeField] private int uiPoolSize = 3;
[SerializeField] private bool enableFade = true;

private AudioSource _musicSource;
```

**Зачем这么多 полей?**
- `config` — каталог звуков (Asset)
- `mixer` — AudioMixer (громкость)
- `sfxPoolSize` — количество SFX-голосов
- `uiPoolSize` — количество UI-голосов
- `enableFade` — глобальный тумблер фейда
- `_musicSource` — источник для музыки
- `_pauseSource` — источник для звука паузы
- `_sfxVoices` — пул SFX-голосов
- `_uiVoices` — пул UI-голосов
- `_lastPlayTimes` — время последнего воспроизведения (для cooldown)
- `_musicSoundId` — текущий SoundId музыки
- `_musicPaused` — флаг паузы музыки

### 2. Awake() — инициализация

```csharp
private void Awake()
{
    // Создаем источник для музыки
    _musicSource = CreateSource("Music", transform);
    _musicSource.loop = true;
    _musicSource.volume = 0f;  // Начинаем с 0 для фейд-ина

    // Создаем источник для звука паузы
    _pauseSource = CreateSource("Pause", transform);
    _pauseSource.loop = true;
    _pauseSource.volume = 0f;
    _pauseSource.playOnAwake = false;

    // Создаем пул SFX-голосов
    for (var i = 0; i < sfxPoolSize; i++)
        _sfxVoices.Add(new SfxVoice(CreateSource($"Sfx_{i}", transform)));

    // Создаем пул UI-голосов
    for (var i = 0; i < uiPoolSize; i++)
        _uiVoices.Add(new SfxVoice(CreateSource($"Ui_{i}", transform)));

    // Восстанавливаем сохранённую громкость
    ApplyVolumeFromPrefs();
}
```

**Почему `_musicSource.volume = 0f`?**
- Начинаем с 0 для фейд-ина
- При старте музыки — плавно увеличиваем до 1
- **Альтернатива:** Установить 1 — но тогда нет плавного входа

**Почему `playOnAwake = false`?**
- Чтобы звук не играл при загрузке сцены
- Играем только когда нужно

### 3. Метод Play()

```csharp
public void Play(string soundId, float? pitch = null, float? pitchRandomRange = null)
{
    // 1. Ищем определение звука в каталоге
    if (config == null || !config.TryGet(soundId, out var definition))
        return;

    // 2. Проверяем есть ли клипы
    if (definition.Clips == null || definition.Clips.Length == 0)
        return;

    // 3. Проверяем cooldown (не слишком ли часто?)
    var playClock = GetPlayClock(definition.Channel);
    if (definition.Cooldown > 0f
        && _lastPlayTimes.TryGetValue(soundId, out var lastTime)
        && playClock - lastTime < definition.Cooldown)
        return;

    // 4. Проверяем allowOverlap (можно ли играть поверх?)
    if (!definition.AllowOverlap && IsPlaying(soundId))
        return;

    // 5. Воспроизводим
    PlayDefinition(definition, pitch, pitchRandomRange);
    _lastPlayTimes[soundId] = playClock;
}
```

**Разбор по шагам:**

| Шаг | Что делает | Зачем |
|-----|-----------|-------|
| 1 | `config.TryGet()` | Ищем SoundDefinition по SoundId |
| 2 | `definition.Clips.Length == 0` | Проверяем, есть ли клипы |
| 3 | `definition.Cooldown > 0f` | Проверяем cooldown |
| 4 | `!definition.AllowOverlap` | Проверяем, можно ли играть поверх |
| 5 | `PlayDefinition()` | Воспроизводим звук |
| 6 | `_lastPlayTimes[soundId] = playClock` | Запоминаем время воспроизведения |

> **Важно:** `cooldown` — минимальный интервал между воспроизведениями одного и того же звука.

### 4. Метод PlayDefinition() — выбор пула

```csharp
private void PlayDefinition(SoundDefinition sound, float? pitch, float? pitchRandomRange)
{
    // Музыка — отдельный путь
    if (sound.Channel == AudioChannel.Music)
    {
        PlayMusic(sound, pitch, pitchRandomRange);
        return;
    }

    // SFX или UI — пул голосов
    var pool = sound.Channel == AudioChannel.UiSfx ? _uiVoices : _sfxVoices;
    var voice = AcquireVoice(pool, sound);
    if (voice == null)
        return;

    PlayOnVoice(voice, sound, pitch, pitchRandomRange);
}
```

**Почему отдельный путь для музыки?**
- Музыка — один источник (`_musicSource`)
- SFX — пул голосов (`_sfxVoices`, `_uiVoices`)
- **Зачем?** Чтобы музыка не прерывалась SFX

---

### 5. Метод AcquireVoice() — поиск свободного голоса

```csharp
private SfxVoice AcquireVoice(List<SfxVoice> pool, SoundDefinition sound)
{
    // 1. Ищем свободный голос
    var voice = FindFreeVoice(pool, sound);
    if (voice != null)
        return voice;

    // 2. Если все заняты — убиваем самый низкоприоритетный
    if (TryStopLowestPrioritySfx(sound.Priority))
        return FindFreeVoice(pool, sound);

    // 3. Нет места — не играем
    return null;
}
```

**Почему `TryStopLowestPrioritySfx()`?**
- Если все голоса заняты — убиваем самый низкоприоритетный
- **Зачем?** Чтобы высокоприоритетные звуки не прерывались
- **Приоритеты:** SFX (50), UI (100), Music (0)

### 6. Метод FadeToAsync() — плавное изменение громкости

```csharp
private static async UniTask FadeToAsync(
    AudioSource source,
    float target,
    float duration,
    CancellationToken ct)
{
    if (duration <= 0f)
    {
        source.volume = target;
        return;
    }

    var start = source.volume;
    var t = 0f;
    while (t < duration)
    {
        await UniTask.Yield(PlayerLoopTiming.Update, ct);
        t += Time.unscaledDeltaTime;
        source.volume = Mathf.Lerp(start, target, t / duration);
    }

    source.volume = target;
}
```

**Почему `Time.unscaledDeltaTime`?**
- `Time.deltaTime` — учитывает паузу (pauseScale)
- `Time.unscaledDeltaTime` — НЕ учитывает паузу
- Нам нужно, чтобы fade работал даже при паузе

**Почему `Mathf.Lerp(start, target, t / duration)`?**
- `t = 0` → `volume = start`
- `t = duration` → `volume = target`
- **Альтернатива:** `Mathf.MoveTowards` — но Lerp проще для понимания

## ✅ Шаблон для копирования

```csharp
public class MyAudioManager : MonoBehaviour
{
    [SerializeField] private AudioSource _musicSource;
    [SerializeField] private int sfxPoolSize = 8;
    private readonly List<AudioSource> _sfxVoices = new();
    
    private void Awake()
    {
        // Создаем пул SFX-голосов
        for (var i = 0; i < sfxPoolSize; i++)
        {
            var go = new GameObject($"Sfx_{i}");
            var source = go.AddComponent<AudioSource>();
            source.playOnAwake = false;
            source.spatialBlend = 0f;  // 2D-звук
            _sfxVoices.Add(source);
        }
    }
    
    public void PlaySound(string clipName)
    {
        var voice = FindFreeVoice();
        if (voice != null)
            voice.clip = Resources.Load<AudioClip>(clipName);
            voice.Play();
    }
    
    private AudioSource FindFreeVoice()
    {
        foreach (var voice in _sfxVoices)
        {
            if (!voice.isPlaying)
                return voice;
        }
        return null;
    }
}
```
