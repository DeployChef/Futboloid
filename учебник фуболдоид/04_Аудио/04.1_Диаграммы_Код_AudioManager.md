## 🔄 Диаграмма жизненного цикла звука

```mermaid
sequenceDiagram
    participant Bus as IGameEventBus
    participant AS as AudioService
    participant AM as AudioManager
    participant SV as SfxVoice
    participant Source as AudioSource
    
    Bus->>AS: Publish(BallContactEvent)
    AS->>AS: FindFreeVoice()
    AS->>AM: AcquireVoice()
    AM->>SV: Find free voice
    SV-->>AM: voice
    AM->>Source: Play clip
    Source->>Source: FadeIn (если enableFade)
    Source-->>AS: Звук играет
    Source->>Source: Окончил воспроизведение
    Source-->>SV: isPlaying = false
```

---

## 🔄 Диаграмма Fade In/Out

```mermaid
sequenceDiagram
    participant AM as AudioManager
    participant Source as AudioSource
    participant CT as CancellationToken
    
    AM->>Source: volume = 0f
    AM->>Source: Play()
    AM->>CT: new CancellationTokenSource()
    AM->>CT: FadeToAsync(source, 1f, duration, ct)
    
    loop Каждые Update
        CT->>Source: volume = Lerp(0f, 1f, t/duration)
    end
    
    Source->>Source: volume = 1f
    Source->>AM: Fade complete
```

## 🔄 Диаграмма паузы

```mermaid
stateDiagram-v2
    [*] --> Playing
    Playing --> FadingOut: PauseMusic()
    FadingOut --> MusicPaused: volume = 0f
    MusicPaused --> MusicResumed: ResumeMusic()
    MusicResumed --> FadingIn: FadeIn to previous volume
    FadingIn --> Playing: volume = targetVolume
    
    note right of Playing
        Музыка играет
        SFX играют
    end note
    
    note right of FadingOut
        Громкость падает до 0
        SFX останавливаются
    end note
    
    note right of MusicPaused
        Музыка на Pause
        Играет звук паузы
    end note
    
    note right of FadingIn
        Громкость растёт
        до предыдущего уровня
    end note
```
