## 4.1 Общая архитектура

Аудио-система разделена на **4 уровня**:

```
Геймплейный код
  ↓ (публикует событие)
Шина событий (IGameEventBus)
  ↓ (слушает события)
AudioService          — связывает события с SoundId
  ↓ (вызывает Play/Stop)
IAudioManager
  ↓
AudioManager          — движок звука
  ↓
AudioSource (Unity)   — физическое воспроизведение
```

### Компоненты

| Компонент | Файл | Роль |
|-----------|------|------|
| `IAudioManager` | `Futboloid.Core/Audio/IAudioManager.cs` | Интерфейс (что умеет) |
| `AudioManager` | `Futboloid.Main/Audio/AudioManager.cs` | Движок звука (как делает) |
| `AudioService` | `Futboloid.Core/Audio/AudioService.cs` | Связующее звено (что играть) |
| `SoundDefinition` | `Futboloid.Core/Audio/SoundDefinition.cs` | Конфиг одного звука |
| `AudioCatalog` | `Futboloid.Core/Audio/AudioCatalog.cs` | Файл-конфиг (Asset) |
| `AudioChannel` | `Futboloid.Core/Audio/AudioChannel.cs` | Перечисление каналов |

## 4.2 IAudioManager — интерфейс

```csharp
public interface IAudioManager
{
    void Play(string soundId, float? pitch = null, float? pitchRandomRange = null);
    void Stop(string soundId);
    void StopMusic();
    void PauseMusic();
    void ResumeMusic();
    void StopAll();
    bool IsPlaying(string soundId);
    void SetMusicVolume(float volume);
    void SetSfxVolume(float volume);
    float GetMusicVolume();
    float GetSfxVolume();
}
```

**Зачем интерфейс?**
- Любой код использует `IAudioManager` — не привязан к реализации
- Можно подменить на тестовый (`NullAudioManager`)
- DI-контейнер знает, что зарегистрировать

## 4.3 AudioManager — движок звука

### Пулы AudioSource

```
AudioManager (GameObject)
├── _musicSource         (1 AudioSource — музыка)
├── _pauseSource         (1 AudioSource — звук паузы)
├── SfxVoice[0..7]       (8 AudioSource — SFX)
│   ├── Source
│   ├── SoundId
│   ├── Priority
│   └── FadeCts
└── UiVoice[0..2]        (3 AudioSource — UI)
    ├── Source
    ├── SoundId
    ├── Priority
    └── FadeCts
```

### Метод Play()

```csharp
public void Play(string soundId, float? pitch = null, float? pitchRandomRange = null)
{
    // 1. Ищем определение звука в каталоге
    if (config == null || !config.TryGet(soundId, out var definition))
        return;

    // 2. Проверяем cooldown
    if (definition.Cooldown > 0f && IsPlayingTooSoon(soundId))
        return;

    // 3. Воспроизводим
    PlayDefinition(definition, pitch, pitchRandomRange);
}
```

### Fade In / Fade Out

```csharp
private static async UniTask FadeToAsync(
    AudioSource source,
    float target,
    float duration,
    CancellationToken ct)
{
    var start = source.volume;
    var t = 0f;
    while (t < duration)
    {
        await UniTask.Yield(PlayerLoopTiming.Update, ct);
        t += Time.unscaledDeltaTime;
        source.volume = Mathf.Lerp(start, target, t / duration);
    }
    source.volume = target;
}
```

## 4.4 AudioService — связующее звено

```csharp
public sealed class AudioService : IDisposable
{
    private readonly IGameEventBus _bus;
    private readonly IAudioManager _audio;
    private readonly List<IDisposable> _subscriptions = new();

    public AudioService(IGameEventBus bus, IAudioManager audio)
    {
        _subscriptions.Add(_bus.Subscribe<BallContactEvent>(OnBallContact));
        _subscriptions.Add(_bus.Subscribe<GoalScoredEvent>(OnGoalScored));
        // ... и так далее для всех событий
    }
}
```

### Маппинг событий на звуки

| Событие | SoundId | Когда играет |
|---------|---------|-------------|
| `BallContactEvent` (Wall) | `BallHit` | Отскок от стены |
| `BallContactEvent` (Keeper/Defender) | `BallHitMan` | Отскок от вратаря/защитника |
| `GoalScoredEvent` (IsPlayerGoal=true) | `GoalScored` | Игрок забил |
| `GoalScoredEvent` (IsPlayerGoal=false) | `GoalConceded` | Противник забил |
| `NavigationChangedEvent` → OnField | `MusicMatch` | Вход на поле |
| `MatchEndedEvent` | `MatchEnd` + stop music | Конец матча |

## 4.5 Добавление нового звука

### Пошаговая инструкция

**Шаг 1.** Положи файлы в `Assets/_Projects/Audio/`

```
Assets/_Projects/Audio/SFX/MyNewSound/
├── sound1.wav
├── sound2.wav
└── sound3.wav
```

**Шаг 2.** Открой `AudioCatalog.asset`

**Шаг 3.** Добавь новую строку в Sounds

| Поле | Значение |
|------|----------|
| Id | `MyNewSound` |
| Channel | `GameplaySfx` |
| Clips | Перетащи 3 файла из папки |
| Mixer Group | Перетащи группу из AudioMixer |
| Priority | `50` |
| AllowOverlap | `true` |
| Cooldown | `0.3` |
| Loop | `false` |
| EnableFade | `false` |

**Шаг 4.** Подпишись на событие в AudioService

```csharp
// В конструкторе
_subscriptions.Add(_bus.Subscribe<MyNewEvent>(OnMyNewEvent));

// Метод
private void OnMyNewEvent(MyNewEvent e)
{
    _audio.Play(AudioCatalog.Ids.MyNewSound);
}
```

**Шаг 5.** Публикуй событие из геймплейного кода

```csharp
_bus.Publish(new MyNewEvent(/* данные */));
```
