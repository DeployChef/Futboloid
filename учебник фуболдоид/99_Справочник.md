## 🚀 Быстрый старт

### Как запустить проект

```
1. Открой Unity
2. Открой сцену Root.unity
3. Нажми Play
4. Игра запустится
```

### Как добавить звук

```
1. Положи файлы в Assets/_Projects/Audio/SFX/MySound/
2. Открой AudioCatalog.asset
3. Добавь новую строку в Sounds
4. Подпишись в AudioService
5. Публикуй событие из геймплейного кода
```

### Как добавить перк

```
1. Создай PerkDefinition (ScriptableObject)
2. Добавь ID в PerkIds.cs
3. Зарегистрируй в RunProgressionService
4. Подключи к BonusPickCoordinator
```

## 📦 Шаблоны кода

### Создание события

```csharp
public readonly struct MyEvent
{
    public string Message { get; }
    public int Value { get; }
    
    public MyEvent(string message, int value)
    {
        Message = message;
        Value = value;
    }
}
```

### Подписка на событие

```csharp
public class MyService
{
    private readonly IGameEventBus _bus;
    private readonly IDisposable _subscription;
    
    public MyService(IGameEventBus bus)
    {
        _bus = bus;
        _subscription = _bus.Subscribe<MyEvent>(OnMyEvent);
    }
    
    private void OnMyEvent(MyEvent e)
    {
        Debug.Log(e.Message);
    }
    
    public void Dispose()
    {
        _subscription.Dispose();
    }
}
```

### Публикация события

```csharp
public class MyPublisher
{
    private readonly IGameEventBus _bus;
    
    public MyPublisher(IGameEventBus bus)
    {
        _bus = bus;
    }
    
    public void DoSomething()
    {
        _bus.Publish(new MyEvent("Hello", 42));
    }
}
```

### Регистрация в DI

```csharp
protected override void Configure(IContainerBuilder builder)
{
    // Singleton
    builder.Register<IMyService, MyService>(Lifetime.Singleton);
    
    // Transient
    builder.Register<ITempService, TempService>(Lifetime.Transient);
    
    // Из сцены
    builder.RegisterComponentInScene<MyComponent>(gameObject.scene);
}
```

### Fade In / Fade Out

```csharp
private async UniTask FadeToAsync(
    AudioSource source,
    float target,
    float duration,
    CancellationToken ct)
{
    var start = source.volume;
    var t = 0f;
    while (t < duration)
    {
        await UniTask.Yield(PlayerLoopTiming.Update, ct);
        t += Time.unscaledDeltaTime;
        source.volume = Mathf.Lerp(start, target, t / duration);
    }
    source.volume = target;
}
```

## 🚫 Антипаттерны

### 1. new() вместо DI

```csharp
// ❌ ПЛОХО
var bus = new GameEventBus();

// ✅ ХОРОШО
var bus = ...  // из конструктора
```

### 2. События как классы

```csharp
// ❌ ПЛОХО
public class MyEvent { public string Message; }

// ✅ ХОРОШО
public readonly struct MyEvent { public string Message { get; } }
```

### 3. Прямой вызов AudioSource

```csharp
// ❌ ПЛОХО
GetComponent<AudioSource>().Play();

// ✅ ХОРОШО
_audioManager.Play("SoundId");
```

### 4. Забывать Dispose()

```csharp
// ❌ ПЛОХО
_bus.Subscribe<MyEvent>(OnEvent);

// ✅ ХОРОШО
var sub = _bus.Subscribe<MyEvent>(OnEvent);
// ... позже
sub.Dispose();
```

### 5. Циклические зависимости

```csharp
// ❌ ПЛОХО
public class A(IB b) { }
public class B(IA a) { }

// ✅ ХОРОШО
public class A(IGameEventBus bus) { }
public class B(IGameEventBus bus) { }
```
